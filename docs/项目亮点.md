## 项目亮点

### WanxAgent 智能对话平台

- **项目描述** 基于大模型的智能对话平台，集成多智能体协作、知识库检索增强与工具调用能力，提供可部署可扩展的工程化方案
- **技术栈** Python FastAPI LangChain LangGraph Vue3 TypeScript Redis MySQL Milvus Elasticsearch MCP
- **项目细节与亮点**
  - **流式对话** 支持流式输出与断连停止，提升交互体验并减少无效消耗
  - **多智能体** 支持按对话配置初始化智能体，适配不同角色与不同任务场景
  - **知识库** 支持多格式文档解析与分块入库，覆盖常见业务资料与长文档场景
  - **混合检索** 结合向量检索与关键词检索，并通过重排序提升召回与准确性
  - **工具扩展** 支持接口规范生成自定义工具，并对接 MCP 服务器生态扩展外部能力
  - **长期记忆** 引入语义记忆检索与更新维护，增强多轮对话一致性与个性化效果
  - **工程落地** 提供启动初始化、鉴权与健康检查等底座能力，支持一键启动降低上手成本
